# FRAIM for Cursor

FRAIM turns AI agents into an accountable workforce through phased jobs, reusable skills, rules, evidence, mentoring, and learning loops.

This Cursor plugin adds:

- A FRAIM rule for discovery and execution.
- A FRAIM skill for job, skill, and rule workflows.
- A `/fraim` command entry point.
- A remote MCP server connection to `https://fraim.wellnessatwork.me/mcp`.

Users authenticate through FRAIM's existing OAuth setup flow. This plugin does not store static API keys or bearer tokens.
